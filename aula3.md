Apresentação em ppt do 3º encontro do nosso grupo de estudos sobre automação industrial, na comunidade dos lab365, no discord, falaqmos sobre as linguagens 
de programação na indústria e o que podemos encontrar por aí.
1. Ladder Diagram (LD)
Onde é usada: Programação de CLPs em processos discretos

Vantagens:

Interface intuitiva (similar a diagramas elétricos)
Ideal para lógicas de comando simples
Amplamente adotada em indústrias
2. Texto Estruturado (ST)
Onde é usada: Sistemas complexos e cálculos avançados

Vantagens:

Sintaxe similar a Pascal/C
Permite algoritmos sofisticados
Melhor legibilidade para programas extensos
3. Function Block Diagram (FBD)
Onde é usada: Controle de processos contínuos

Vantagens:

Representação visual de funções
Facilita a reutilização de código
Ideal para controle PID
4. Sequential Function Chart (SFC)
Onde é usada: Processos com etapas bem definidas

Vantagens:

Organiza operações em sequências lógicas
Excelente para máquinas de estados
Simplifica a manutenção
5. Instruction List (IL)
Onde é usada: Otimização de código em hardware limitado

Vantagens:

Baixo consumo de memória
Execução extremamente rápida
Similar ao assembly
